=================================================================================
◆ Discord Rich Presence (Wodistant用プラグイン) - Ver1.0.0 (2025/03/05)
=================================================================================

■概要

  Wodistantに「Discordのアクティビティステータス（Rich Presence）を表示させる」機能を追加するプラグインです。
  サーバーの皆にウディタでゲームを開発中なことをアピールしちゃいましょう。

■導入方法

  同梱の「WodistantDiscordRP.dll」を、Wodistantの「plugins」フォルダに入れてください。
  Wodistantを起動後、メニューバーの「ヘルプ」->「読み込まれたプラグイン一覧」で、
  「Discord Rich Presence」が表示されていたら成功です。

  Discordクライアントを起動している状態でウディタに接続すると、Disocrdにアクティビティが表示されます。
