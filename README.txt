=================================================================================
◆ Discord Rich Presence (Wodistant用プラグイン) - Ver1.0.0 (2025/03/05)
=================================================================================

■概要

  Wodistantに「Discordのアクティビティステータス（Rich Presence）を表示させる」機能を追加するプラグインです。
  サーバーの皆にウディタでゲームを開発中なことをアピールしちゃいましょう。

■導入方法

  同梱の「WodistantDiscordRP.dll」を、Wodistantの「plugins」フォルダに入れてください。

  【重要】高確率でセキュリティ機能（MOTW）にブロックされているので、次の手順で解除してください（そうしないとプラグインが認識されません）
    ①「WodistantDiscordRP.dll」を右クリックして「プロパティ」を選択
    ②下部に「セキュリティ: このファイルは～」と表示されていたら、右の「許可する」にチェックを入れて「OK」を押す
  （何も表示されていなければそのままで大丈夫です）

  Wodistantを起動後、メニューバーの「ヘルプ」->「読み込まれたプラグイン一覧」で、
  「Discord Rich Presence」が表示されていたら完了です。

■使い方

  Discordクライアントを起動している状態でウディタに接続するだけです。
  これで自動的にDiscordにアクティビティが表示されます。
